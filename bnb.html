A full-stack home-rental system built with Angular/Ionic, Node.js, Express, and MySQL.

July 2019

Overview

For my Full-Stack Coding course for iXperience, our project was to create a functioning app that replicated the services of Airbnb. I created 3 different applications:

  1. Consumer mobile app - allows users to join, view listed properties, and book stays.
  2. Provider mobile app - allows users to join and list & manage their own properties for rent.
  3. Admin panel web app - allows admins to view all registered app users, listings and booking requests.

Tech Stack

The consumer and provider apps are build with Angular's Ionic framework, with Node.js for the backend. The admin panel is built with Angular. The API is created with Express with a MySQL database.

Description

For the Consumer and Provider applications, users are able to register or login to an account. All users are authenticated by the API using JWT (JSON Web Tokens).
